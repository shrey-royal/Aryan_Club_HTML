Marquee Tag Attributes Demonstration

bgcolor: This sets the background color of the marquee.

This is a marquee with a background color.

width: This sets the width of the marquee.

This marquee is 50% of the page width.

height: This sets the height of the marquee.

This marquee has a height of 50px.

direction: This sets the direction of the scrolling content (left, right, up, down).

This marquee scrolls to the right. This marquee scrolls up.

behavior: This sets the scrolling behavior (scroll, slide, alternate).

This marquee scrolls continuously (default). This marquee slides and stops at the end. This marquee bounces back and forth.

loop: This sets the number of times the marquee scrolls. (-1 for infinite)

This marquee scrolls 2 times only.

scrollamount: This sets the speed of scrolling (higher value = faster).

This marquee scrolls faster. This marquee scrolls slower.

scrolldelay: This sets the delay between scrolls in milliseconds.

This marquee has a delay of 200ms.

hspace: This sets the horizontal margin for the marquee.

This marquee has horizontal spacing of 50px.

vspace: This sets the vertical margin for the marquee.

This marquee has vertical spacing of 20px.

vspace: This sets the vertical margin for the marquee.
